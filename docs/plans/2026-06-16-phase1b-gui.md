# AgentPipe Phase 1b 实施计划:Tauri 壳 + UI + 运行时控制

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把 Phase 1a 的 headless 引擎装进 Tauri 2 桌面应用,提供编排页 + 运行控制台,并让运行时控制(gate 批准/跳过、Abort 杀进程、失败决策 gate)真正接通。

**Architecture:** 引擎在后台线程跑,事件经转发线程 `app.emit` 给 webview,webview 经 `invoke` 回送指令到引擎 channel。引擎仅小改(serde 派生 + Control 句柄)。前端 React + Vite,事件归约成步骤树。

**Tech Stack:** Tauri 2,Rust 1.93,React 18 + TypeScript + Vite,`@tauri-apps/api` v2,`tauri-plugin-dialog`,`libc`(Unix 进程组杀),Vitest(reducer 单测)。

- [ ] **Step 1: bridge.rs —— 启动引擎 + 转发线程**

```rust
use std::sync::mpsc;
use std::thread;
use agentpipe_engine::executor::{Executor, RunnerBins};
use agentpipe_engine::manifest::Manifest;
use agentpipe_engine::protocol::{Command, Event};
use tauri::{AppHandle, Emitter};

pub const EVENT_CHANNEL: &str = "engine://event";

pub struct Started {
    pub commands: mpsc::Sender<Command>,
}

/// 在后台线程跑引擎,事件经转发线程 emit 到 webview。
pub fn start(app: AppHandle, manifest: Manifest, bins: RunnerBins) -> Started {
    let (event_tx, event_rx) = mpsc::channel::<Event>();
    let (cmd_tx, cmd_rx) = mpsc::channel::<Command>();

    // 转发线程:引擎事件 → webview
    thread::spawn(move || {
        for evt in event_rx {
            let _ = app.emit(EVENT_CHANNEL, evt);
        }
        // event_tx 在引擎线程结束时 drop → 本循环退出
    });

    // 引擎线程
    thread::spawn(move || {
        let mut ex = Executor::new(manifest, bins, event_tx, cmd_rx);
        ex.run();
    });

    Started { commands: cmd_tx }
}
```

```rust
fn runner_bins() -> RunnerBins {

        codex: std::env::var("AGENTPIPE_CODEX_BIN").unwrap_or_else(|_| "codex".into()),

#[tauri::command]
pub fn start_run(app: AppHandle, state: State<AppState>, path: String) -> Result<(), String> {
    let mut active = state.active.lock().unwrap();
    if active.is_some() {
        return Err("已有运行中的 Run,请先结束".into());
    }
    let yaml = std::fs::read_to_string(&path).map_err(|e| format!("读取 {path} 失败: {e}"))?;
    let manifest = Manifest::parse(&yaml).map_err(|e| e.to_string())?;
    manifest.validate().map_err(|e| e.to_string())?;

    // Control 在 Task 5 接入;此处先用默认,Task 5 改为传入引擎
    let control = Arc::new(agentpipe_engine::control::Control::default());
    let started = bridge::start(app, manifest, runner_bins());
    *active = Some(ActiveRun { commands: started.commands, control });
    Ok(())
}

#[tauri::command]
pub fn send_command(state: State<AppState>, cmd: Command) -> Result<(), String> {
    let active = state.active.lock().unwrap();
    match active.as_ref() {
        Some(run) => run.commands.send(cmd).map_err(|e| e.to_string()),
        None => Err("没有运行中的 Run".into()),
    }
}
```

```rust
    std::path::PathBuf::from(env!("CARGO_MANIFEST_DIR")).join("../templates")

#[tauri::command]
pub fn list_templates() -> Result<Vec<String>, String> {
    let dir = templates_dir();
    let mut names = vec![];
    for entry in std::fs::read_dir(&dir).map_err(|e| e.to_string())? {
        let p = entry.map_err(|e| e.to_string())?.path();
        if p.extension().and_then(|s| s.to_str()) == Some("yaml") {
            if let Some(stem) = p.file_stem().and_then(|s| s.to_str()) {
                names.push(stem.to_string());
            }
        }
    }
    Ok(names)
}

#[tauri::command]
pub fn load_template(name: String) -> Result<String, String> {
    // 防路径穿越:只允许纯文件名
    if name.contains('/') || name.contains("..") {
        return Err("非法模板名".into());
    }
    let p = templates_dir().join(format!("{name}.yaml"));
    std::fs::read_to_string(&p).map_err(|e| e.to_string())
}
```

```tsx
  const addStep = () => update({ steps: [...m.steps, emptyStep(`step-${m.steps.length + 1}`)] });
  const delStep = (i: number) => update({ steps: m.steps.filter((_, j) => j !== i) });
  const move = (i: number, d: -1 | 1) => {
    const j = i + d;
    if (j < 0 || j >= m.steps.length) return;
    const next = [...m.steps];
    [next[i], next[j]] = [next[j], next[i]];
    update({ steps: next });
  };

  const save = async () => {
    try { await ipc.saveManifest(m, savePath); setErr(null); }
    catch (e) { setErr(String(e)); }
  };

  return (
    <div>
      <input placeholder="任务名" value={m.name} onChange={(e) => update({ name: e.target.value })} />
      <button onClick={async () => { const d = await ipc.pickDir(); if (d) update({ target: d }); }}>
        选 target:{m.target || "(未选)"}
      </button>
      <select value={m.mode} onChange={(e) => update({ mode: e.target.value as Manifest["mode"] })}>
        <option value="step">step(逐步批准)</option>
        <option value="auto">auto(自动跑)</option>
      </select>

      {m.steps.map((s, i) => (
        <StepCard key={i} step={s} onChange={(s2) => setStep(i, s2)}
          onUp={() => move(i, -1)} onDown={() => move(i, 1)} onDelete={() => delStep(i)} />
      ))}
      <button onClick={addStep}>+ 加步骤</button>

      <input placeholder="保存到 task.yaml 路径" value={savePath} onChange={(e) => setSavePath(e.target.value)} />
      <button onClick={save}>保存</button>
      <button onClick={() => onRun(savePath)} disabled={!savePath}>运行</button>
      {err && <div style={{ color: "red" }}>{err}</div>}
    </div>
  );
}
```

```ts
import type { EngineEvent, StepStatus, RunStatus, GateKind } from "../types";

export interface StepView { status: StepStatus; summary?: string; error?: string; }
export interface GateView { step_id: string; suggestion: string; expects_artifact: boolean; gate_kind: GateKind; }
export interface RunState {
  name: string;
  order: string[];                       // 出现顺序
  steps: Record<string, StepView>;
  loops: Record<string, { iteration: number; result?: string }>;
  activeGate: GateView | null;
  runStatus: RunStatus | null;
  log: string[];                         // 人读流水
}

export const initialRunState = (): RunState => ({
  name: "", order: [], steps: {}, loops: {}, activeGate: null, runStatus: null, log: [],
});

function touch(s: RunState, id: string, st: StepView) {
  if (!s.order.includes(id)) s.order.push(id);
  s.steps[id] = { ...s.steps[id], ...st };
}

export function runReducer(prev: RunState, e: EngineEvent): RunState {
  const s: RunState = { ...prev, steps: { ...prev.steps }, loops: { ...prev.loops }, order: [...prev.order], log: [...prev.log] };
  // 任意非 gate 事件清空 activeGate(gate 已被响应)
  if (e.type !== "StepAwaitingGate") s.activeGate = prev.activeGate;
  switch (e.type) {
    case "RunStarted": s.name = e.name; s.log.push(`▶ ${e.name}`); break;
    case "StepStarted": touch(s, e.step_id, { status: "Running" }); s.activeGate = null; break;
    case "StepProgress": s.log.push(`  ${e.line}`); break;
    case "StepAwaitingGate":
      touch(s, e.step_id, { status: "AwaitingGate" });
      s.activeGate = { step_id: e.step_id, suggestion: e.suggestion, expects_artifact: e.expects_artifact, gate_kind: e.gate_kind };
      break;
    case "StepFinished": touch(s, e.step_id, { status: e.status, summary: e.summary }); s.activeGate = null; break;
    case "StepFailed": touch(s, e.step_id, { status: "Failed", error: e.error }); break;
    case "LoopIteration": s.loops[e.loop_id] = { iteration: e.iteration }; break;
    case "LoopConverged": s.loops[e.loop_id] = { iteration: e.iterations, result: "收敛" }; break;
    case "LoopMaxReached": s.loops[e.loop_id] = { iteration: e.max, result: "到上限未干净" }; break;
    case "RunFinished": s.runStatus = e.status; s.activeGate = null; s.log.push(`■ ${e.status}`); break;
  }
  return s;
}
```

```rust
use std::sync::atomic::{AtomicBool, Ordering};
use std::sync::Mutex;

#[derive(Default)]
pub struct Control {
    abort: AtomicBool,
    current_pgid: Mutex<Option<u32>>,
}

impl Control {
    pub fn request_abort(&self) {
        self.abort.store(true, Ordering::SeqCst);
        self.kill_current();
    }
    pub fn is_aborted(&self) -> bool {
        self.abort.load(Ordering::SeqCst)
    }
    pub fn set_current(&self, pgid: Option<u32>) {
        *self.current_pgid.lock().unwrap() = pgid;
    }
    /// 杀掉当前子进程组(连带 shell 起的孙辈),解决 Phase 1a 的 sleep 遗留问题。
    pub fn kill_current(&self) {
        if let Some(pgid) = *self.current_pgid.lock().unwrap() {
            #[cfg(unix)]
            unsafe { libc::killpg(pgid as i32, libc::SIGKILL); }
        }
    }
}
```
